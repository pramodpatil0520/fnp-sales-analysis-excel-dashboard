## About the Project
This is a simple sales analysis dashboard built in Excel to understand how sales are performing across different categories, regions, and time periods.

The goal of this project was to take raw sales data and convert it into meaningful insights using basic data analysis and visualization techniques.

---

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  

---

## Dashboard Features
- Sales trend over time  
- Category-wise performance  
- Region-wise comparison  
- Top products/customers  
- Interactive filters for better analysis  

---

## Why I Made This
I created this project to practice data analysis and improve my skills in building dashboards that are easy to understand and useful for decision-making.

---

## How to Use
1. Download the Excel file from this repository  
2. Open it in Microsoft Excel  
3. Go to the dashboard sheet  
4. Use the slicers to explore different insights  

---

## Key Observations
- A small number of categories contribute to most of the revenue  
- Sales show variation across different months  
- Some regions perform consistently better than others  

---

## Scope of the Project
This project is limited to analyzing historical sales data using Excel. It focuses on descriptive analysis such as trends, comparisons, and performance tracking. The dashboard helps in understanding past data but does not include forecasting or real-time data integration.
